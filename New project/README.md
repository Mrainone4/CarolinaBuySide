# Carolina Buyside Site

Static multi-page website for Carolina Buyside.

## Pages

- `index.html`: homepage
- `about.html`: mission and programming
- `team.html`: officer and leadership page
- `join.html`: application and interest page

## Customize

1. Update the LinkedIn URL if the club adopts a new public page.
2. Revise officer names, sector heads, and bios on `team.html` as leadership changes.
3. Update `join.html` whenever the next recruiting cycle opens.
4. Add semester dates, meeting details, and contact information where needed.

## Deploy

You can deploy this as a static site on:

- GitHub Pages
- Netlify
- Vercel

For all three, upload the whole folder as-is. The homepage should be `index.html`.
